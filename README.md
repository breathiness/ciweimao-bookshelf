## 一、书籍介绍

### （一）青春日常

- 我在东瀛开出租  
我是司机 著  

![enter image description here](https://novel-cdn.kuangxiangit.com/uploads/allimg/c180225/25-02-18234511-38295-100055432.jpg)

简介：我叫藤原拓海，今年18岁，目前在东瀛开出租车。  
说实话，我个人对于自己的车技还是十分自信的，不信你问她们。  
“呕哇——” 
你看，她们感动到吐了。  

标签：八万八 搞笑 老司基 恋爱 青春 日常  
简评：其实我对《我在东瀛XXX》这一类的综漫小说是有些小偏见的。虽然是有写的很棒的作品，但在大多数情况下都是套路满满。主角穿越当文抄公，然后与几位书客劳模搞搞暧昧……  
而这一本还是相对来说比较有新意的。主角是著名老司机藤原拓海，工作是开出租车。难得一见的主角不是日本高中生。  
其他角色写的也挺好。智障警察虽然丢人，但我还是感觉挺有意思的。经常被他智障般的脑回路逗笑。  
感情线方面就目前来说还是比较清晰的。专注于和女主角发狗粮，其他妹子(比如劳模雪之下)都在朋友阶段。这种比较慢悠悠的感情发展我还是比较喜欢。  
作为综漫，混合的世界不是很多，并没有太多相互磨合所带来的违和感。  
剧情方面字数差不多20W，仅仅是开了个头。不过看着还不错，虽然也是本质也是熟悉的装逼打脸，但是特别让人意外的展现方法也是挺有趣。出人意料的操作带来的惊讶，再加上一些奇妙的幽默感。让我想起当年看《的士速递》时的感觉。说起来前面的剧情也真的是有一些来自《的士速递》。  
总结一下，这是一本非常有意思的搞笑综漫小说。目前字数不多，不过有很大的潜力。  
更新状态：连载中  

### （二）异界幻想

- 我的大宝剑  

学霸殿下 著  

![enter image description here](https://novel-cdn.kuangxiangit.com/uploads/allimg/c160701/01-07-16220323-6809-100011781.jpg)

简介：始皇历1838年，天元战争结束，一个刚穿越就被逼着打了六年的仗的男人将手中的大宝剑一扔，带着灵魂上的伤痛与茫然，摇摇晃晃地离开了满目疮痍的战场，想要去看看这个陌生而神秘的世界，我们的故事，就从他重获自由的两年后开始。

　　然而他发现，事情并没有这么简单，那铁与火交织的六年，已经让他与这个新世界的命运紧紧地纠缠在一起，推行变革的汉家王朝，虎视眈眈的西方列强，挥舞帝兵的武者，操纵机械的枪炮士，驾驭蒸汽机关的工程师，一身致命武装的改造人，在这个科技树歪到姥姥家的新世界，各自处于巅峰状态的东西方文明正等待着一场华丽绚烂的终极碰撞，而他与那把被他扔掉的大宝剑，恰好正是一切的起源与因由。

　　就像命运一样，他躲不过去。

　　而且要命的是，他好像被自己扔掉的那把大宝剑给诅咒了……唔，亦或是祝福也不一定？

　　他体内的大宝剑之力，有一部分变成了大保健之力……这是件很麻烦的事情，其中一个比较严重的后果，就是他永远失去了扶老奶奶过马路的资格。

　　因为现在的他，只要碰触到女性的任何部位，就会让对方……咳咳，他妈的，真是太扯淡了。

　　“大家好，我叫孙朗，最近因为某些原因转业成了一名游侠……还有，也许小时候的我渴望着各种萌妹子和一个大大的后宫，但现在我讨厌H，非常讨厌。”

　　“……不要用这种眼神看着我，这是真的，你们一定要相信我。”

标签：撕字典 家里人知道吗 处男小说 女装大佬 (这都什么鬼标签)  
简评：作为书客台柱子，这本书的优秀程度想必也是不需要多说。各方面都能称得上顶尖水平。  
这次依然是相当的有学霸个人风格的作品，非常不错。平时嘻笑怒骂、欢脱搞笑。玩梗数量巨多，一言不合就开车。但到了每段剧情的关键时刻，真正严肃起来真的能让人感动得热泪盈眶。这也是我如此喜欢学霸作品的最直接的原因。可以说既满足了只想享受欢乐吐槽的读者，也满足了想要认真看剧情的读者。  
剧情方面也非常厉害。精彩得能让人一口气看完，或轻松愉快，或紧张刺激，或诡异恐怖……每段剧情都能让人感慨万千。虽然主角已经无敌，但却不同于一般的无敌流。更多的还是将无敌的力量作为智谋的辅助，而不是所有困难都能靠蛮力一拳打破。  
角色方面我觉得是这本书里写的最好的一部分。尤其是对各配角的描写，“愚不可及”的平一刀、找回初心的孔仲吾、一往情深的王氏和戚冠岩……或是心中的信念，或是对亲人的感情。每个人都让我印象深刻。看着不知不觉就已经热泪盈眶(其实哭出来了)。  
最后想对学霸说一句，在我的眼中这是一本能够抬头挺胸自豪地介绍给家人的小说。  
更新状态：连载中​  

### （三）动漫穿越

- 金毛狮王异闻录

灰白之裔 著

![enter image description here](https://novel-cdn.kuangxiangit.com/uploads/allimg/c181120/20-11-18175345-99918-100034889.jpg)

简介：我写综漫也不为别的，就是为了和喜欢的纸片人角色谈恋爱。 

标签：魔改名著、无限流、脑洞大开、综漫 

简评：相当强势的性|欲型写手。说啪就啪，而且连具体的描写都毫不避讳，熟练地地避开了书客的敏感词，真心爽到。这年头能看到如此正直的作者是真的难得。(听说最近被人举报了一波，不知道能剩多少) 
当然，就算抛开这方面的内容，作品本身的质量也是相当不错。可以说是非常有意思的一个不套路的无限流。 
主角智商长期在线。而且个性相当果断、该杀就杀该啪就啪。偶尔也能看到柔弱的一面。各种个性不同的妹子基本都保持了自己的特色，并不是无脑的后宫+1 
穿越的世界也相当有个性，你见过去《红与黑》的无限流吗？之后还有魔改童话世界、魔改《水浒传》……目前虽然仅仅经历三个世界但每个世界的流程都很长，可以看的很爽。 
对世界的各种魔改真的挺让我惊喜的，让世界变得新奇有趣许多。 
各种梗的插入也非常恰当，是一本能让人一口气看完的有趣作品。非常推荐！ 

更新状态：连载中 

- 离开后被主神再度叫回去的我  

续梦者 著  

![enter image description here](https://novel-cdn.kuangxiangit.com/uploads/allimg/c171124/24-11-17140344-24900-100049740.jpg)

简介：我曾经是轮回者。  
一无所知的我，再度被主神叫了回去——唯一幸存的轮回者。  
那已与过去过去完全不同的，属于选召者的新空间。  
我的记忆、我的过去、我的队友、我的历练····我是燃烧至今的旧时代余烬。  
前三卷，简称···崩崩崩。  

标签：重归无限之人 卡牌 无限流  
简评：前面非常厉害的一本无限流小说，尤其是开场的新手关以及第二场的赌场关，真的是非常精彩。
第一关主角虽然没有曾经无限世界的记忆，但是依然展现出了面对陌生环境的强大适应力以及应有的警惕。与女主二人充分利用身边环境。最后的战斗更是展现出了主角的意志力以及相应的觉悟。  
第二场的赌场关再度与女主相遇并且面对经典的海盗分金式博弈，进行了一场精彩的赌博。在后面还与女主一起上演了一出《无良夫妻行骗记》。  
不过可惜的是在第三场魔法少女伊莉雅篇里作者开始转型，主角智商飞速下滑。与第一场时的主角判若两人。让我想起了王|选篇的486与之前的对比……不过在最后主角也算是得到了一些成长。  
最新正在连载中的最二网管篇主角的外置式智商再次上线。不知道后面会是什么样的发展。还是希望作者能调整好自己的状态。 
这本书对我来说是非常的高开低走，有些失望。  
但总体来说还是不错的，相比起其他的无限流的无脑后宫喜+1，作者对动画人物感情的看法我还是比较赞同的。我觉得这本书还是可以继续观望。  
更新状态：连载中

### （五）战争历史

- 遵命！陛下

林登万233 著  

![enter image description here](https://c1.kuangxiangit.com/uploads/allimg/c210601/01-06-21114929-77594.jpg)

简介：  
　1833年，柏林大学。  
林登万看着眼前的冷傲御姐。  
　她拥有一头柔顺的金色长发，面容姣好，而且有容奶大。  
　短裙与过膝靴的搭配相得益彰，那绝对领域实在惹人瞩目。  
　她的姓名牌上写着：「安娜·冯·俾斯麦」  
　她说：“当代的重大问题，不是演说和决议能够解决的，这些问题唯有铁和血才能解决。”  
本书又名《欢迎来到实力至上的19世纪欧洲》《关于我的宰相和皇后上演宫斗戏这件小事》《俾斯麦大小姐想让我服从？政客们的权利头脑战~》《我的青春治国物语果然有问题》《RE：从零开始的普鲁士生活》《某科学的条顿骑士团》《德意志战记》《掀起波兰》《绿罗狂喜》《今日汝髪》《全球收狗：大嘤帝国》《沙雕鹅国》  

简评：  
这是一本很不错的历史类种田文。阅读起来非常流畅。  
这本书主要讲述了穿越者德皇林登万的一生。看着主角以穿越者的身份推动社会与科技的进步、将普鲁士一步步推进到一个统一的德意志帝国。跟各种有趣的历史人物交流、对抗。确实相当有看种田文独特的快乐。  
当然，看这本书还有一些很有趣的小乐趣。比如作为跟主角一样的“反美魔怔人”，看着主角在文中的反美行为确实非常的快乐。  
魔改后的历史人物也是有各自的特色。与历史有不同，又有相似之处。  
这个世界虽然通过魔改，增加了魔能骑士的设定。但是这个设定其实并没有给历史的发展带来特别大的变化。主要其实还是靠主角团从政治方面带来变化。在剧情发展过程中也显得挺没有存在感的。不得不说是有点小小的遗憾。  

### （六）推理灵异

- ROOT超级用户权限  

二重未知 著  

![enter image description here](https://novel-cdn.kuangxiangit.com/uploads/allimg/c161021/21-10-16014411-60434.jpg)

简介：在一九八五年五月获耶路撒冷文学奖时在典礼中的演讲词，米兰·昆德拉使用了这句话：
“人类一思考，上帝就发笑。”
这句话的意思是：多思考，逗上帝他老人家开心。不然人迟早要倒霉！（并不是）

标签：推理
简评：BCEFGADEFGDEFDEFABCDEF  
这是一本相当有意思的解谜、推理类小说。也是我第一本加了读者群的小说(虽然只在刚进去的时候聊了聊解谜游戏)。  
这本书可以说是让我“玩”的最开心一本书了。里面的各种谜题非常的烧脑有趣，最能享受这本书乐趣的阅读方法，是自己先把里面的谜题尝试解一遍，然后再对照答案。有种玩解谜游戏的感觉。除了答案，作者也给出了解答的方法以及相关的科普。  
仅仅第一卷就已经让我深陷其中，Linux、ASCII码、摩尔斯电码……各种计算机相关的谜题和梗让我看的相当爽。而且这些谜题都是关于“开始”，可以说是完美契合第一卷的主题。强大神秘的主办方、三年前的黑幕……都让人非常好奇。人物虽然多并且也有自己的特色，不过写的并不特别鲜活。  
随着接下来的展开，root权限的使用、各角色间的矛盾冲突都愈发激烈，看的人欲罢不能。不过没有第一卷那么多的谜题了，还是挺伤心的……  
之前我看到了第三卷，感觉那一段的剧情稍微有点乱，再加上当时我状态也有点糟。就开着自动订阅养着，现在已经更新到第八卷了，接下来可以看个爽。  
更新状态：连载中(不稳定，但是还算不错)

不死人 著

![enter image description here](https://novel-cdn.kuangxiangit.com/uploads/allimg/c170815/15-08-17165937-66014-100042074.jpg)

简介：一本超好看的游戏小说！  

标签：战斗、朋克、网游  

简评：坑树的新作，质量依然很不错。最近坑树似乎渐渐开始填坑了，这本的更新也稳定下来了，或许可以追。这本书依然是在我这里有评分加成的单元剧类型。再加上有点赛博朋克的主世界观、极其奇特的游戏世界，确实非常吸引人。已经结束的第一段剧情是蒸汽朋克风城市，也是极其的带感。场景描写的相当生动，各种细节上的描写将城市的风貌很好的展现出来。正在进行的第二段剧情又到了非常日系的比良坂……游戏中各种风格迥异的地方、非常精彩的剧情以及非常“玩家”心态的主角都非常让人期待后续的发展。  
目前看来坑树似乎进入了满节操爆肝模式，​最近甚至日常十更。而作品质量大部分也是保持着很高的水准。游戏中世界观的庞大远超我一开始的想象，而最让我惊喜的还是关于现实的描绘，原本以为只是作为一个背景般的存在，没想到还认真的描绘出赛博朋克风的未来都市，各种细节的展示都非常真实。月面旅游那段是真的很赞。  

更新状态：已完结

### （十）超现实都市

- 真是抱歉，好不容易觉醒的超能力却没什么用

寂灭血觋 著

![enter image description here](https://novel-cdn.kuangxiangit.com/uploads/allimg/c171204/04-12-17125626-15590-100051062.jpg)

简介：
在6月9日上午10点23分12秒被称为“进化之时”的事件当中，全体的七十亿人类全部昏迷了2.14秒。等到他们醒来之后，人类全部拥有了超能力。但是可惜的是这些超能力却没有什么用，只是一堆莫名其妙的现象而已。在“进化之时”的三个月后，16岁的高中生陆平在家门口遇见了莫名其妙的少女：“我想要组建超能力战队来征服世界。请问你有兴趣吗？”少年完全莫名的超能力战斗物语就此开始！  

标签：战斗、校园、轻小说、社团  

简评：全世界所有人都获得了各不相同的超能力，但这些超能力都有同一个特点：“没卵用”。
整个故事开始于拥有“镜面反射”的主角被一位神秘的凉宫春日型美少女找上，这种套路到不行的开头。  
然而，这部作品却显得相当的有趣。装傻系女主角加上吐槽系男主角的组合效果确实很棒，日常方面挺欢乐的。
