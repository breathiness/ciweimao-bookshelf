## 零、前言
这个项目主要用于记录我在刺猬猫看过的书以及相应的点评；  
就在前几天我在刺猬猫的所有的帖子全部被删除，并且被告知无法恢复。只剩下书单还在；  
原因是阿里云检测到我有严重违规的言论。但是因为查不到是我的哪一条言论有问题，于是刺猬猫的审核人员就把我在刺猬猫4年间发的一万多条评论全部删除了。
本来  
今天开始把这个书单的点评都补完，然后在书客4年的历程，大概就算是宣告完结了。
